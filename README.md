# Timber

Аркадная 2D-игра на Rust, вдохновлённая классическим Timberman. Игрок стоит у дерева и уворачивается от падающих веток, переключаясь между левой и правой сторонами.

## Геймплей

Ветки появляются сверху и летят вниз, ускоряясь со временем. Задача — не попасть под ветку, переключаясь влево/вправо. Игра заканчивается при столкновении с веткой или по истечении таймера.

## Управление

| Клавиша | Действие |
|---------|----------|
| `←` | Перейти влево |
| `→` | Перейти вправо |
| `Left Shift` | Пауза / продолжить |
| `Enter` | Начать заново |

## Стек

- **Язык:** Rust (edition 2024)
- **Фреймворк:** [ggez](https://github.com/ggez/ggez) 0.9
- **Рандом:** rand 0.9

## Структура проекта

```
src/
├── main.rs              — точка входа, настройка окна и шрифтов
├── game/
│   └── mod.rs           — игровой цикл, состояние, отрисовка HUD
├── motion/
│   └── mod.rs           — горизонтальное движение (скорость, направление)
└── object/
    ├── mod.rs
    ├── player/
    │   └── mod.rs       — игрок, переключение сторон, коллизия
    ├── tree/
    │   └── mod.rs       — дерево, ветки, спавн, ускорение
    └── scenic/
        └── mod.rs       — декоративные объекты (облака, пчела)

resources/
├── graphics/            — спрайты (background, tree, branch, player, ...)
└── fonts/               — шрифт KOMIKAP
```

## Сборка и запуск

```bash
cargo run
```

Окно запускается в разрешении 1920x1080. Ресурсы загружаются из директории `resources/`.

## Механики

- **Ускорение:** скорость веток увеличивается каждые 20 секунд, интервал спавна автоматически подстраивается, чтобы на экране всегда было до 3 веток
- **Антиповтор:** не более 2 веток подряд на одной стороне
- **Коллизия:** AABB-проверка с учётом трансформаций спрайтов (rotation, scale)
